# Number Guessing Game 🎮

A simple C-based number guessing game where the player has 5 chances to guess random numbers between 15–20. Win by guessing correctly at least 3 times!

## How to Play
1. Run the program.
2. Enter your name.
3. Choose whether to view the rules.
4. Guess numbers between 15–20 for 5 rounds.
5. If you guess correctly 3 or more times, you win!

## Features
- Random number generation
- User input handling
- Win/lose condition tracking
